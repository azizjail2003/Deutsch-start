# Deutsch Start

A free, beginner-first website for learning **German A1**. A structured
65-lesson roadmap (grouped into six units) paired with the best free resources
for grammar, vocabulary, listening, speaking, writing and exam practice.

This is a standalone companion to *Road to Dortmund* — same trusted resources,
but a general-purpose German learning hub with its own identity.

## Features

- Full A1 roadmap: 65 lessons with grammar focus and vocabulary, grouped into 6 units
- Lesson explorer with live search and unit filtering
- Six skill cards linking curated free resources (DW Nicos Weg, Easy German,
  YouGlish, Vocaroo, LanguageTool, Duolingo, Goethe and more)
- Light / dark mode, fully responsive, keyboard-accessible, reduced-motion aware
- No backend, no account, no tracking

## Local development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
pnpm typecheck
pnpm build
```

## Deploy

Import the folder into Vercel (framework auto-detected as Next.js). No
environment variables or backend services are required.
